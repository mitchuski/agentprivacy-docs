# 0xagentprivacy Documentation Suite

Privacy-First AI Agent Architecture for Human Sovereignty

## Quick Start

### View Documentation Locally

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Start the server:
```bash
python server.py
```

3. Open your browser:
```
http://localhost:7000
```
